# Elif & Muharrem — Engagement Invitation (Dijital Davetiye)

A modern, elegant, and interactive digital engagement invitation webpage built with vanilla HTML, custom CSS3, and JavaScript. Optimized specifically for mobile devices and fast load times.

## 🚀 Live Demo
You can view the live site at: **[davetiye-taupe-beta.vercel.app](https://davetiye-taupe-beta.vercel.app)**

---

## ✨ Features

- **🌸 Interactive Opening Experience**: Classic keten ip (twine) and envelope sealing design. Users click the envelope to animate and reveal the invitation letter.
- **🎵 Ambient Audio**: Interactive background music player with dynamic play/pause control and state retention.
- **🍃 Particle & Letter Animations**: HTML5 Canvas-based floating envelopes and delicate background drift animation effects.
- **📱 Fully Responsive Design**: Built mobile-first, ensuring high visual quality and readable typography across iOS, Android, and desktop screens.
- **🎨 Premium Typography**: Curated font sets from Google Fonts (Cormorant Garamond, Cinzel, Great Vibes, EB Garamond) to deliver a classic paper-like aesthetic.
- **⚡ Lightweight & Performant**: No complex frameworks. Coded in vanilla JS and raw CSS with optimized performance.

## ⚙️ Running Locally

Since this is a static webpage, you can run it directly:

1. Clone this repository:
   ```bash
   git clone https://github.com/developermuhammed98/davetiye.git
   cd davetiye
   ```
2. Open `index.html` in any web browser, or serve it using an editor extension like VS Code Live Server.
